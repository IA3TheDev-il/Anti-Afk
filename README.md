## Run Locally

Clone The Project

```bash
  git clone https://github.com/IA3TheDev-il/Anti-Afk
```

Install Dependencies

```bash
  pip3 install -r requirements.txt
```

Go To The Project Directory

```bash
  cd Anti-Afk
```

Start The Program

```bash
  python main.py
```
